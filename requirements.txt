# ============================================================================
# OSINT Investigator v3 — Python dependencies (the CORE engine)
# ============================================================================
# These are the libraries the system itself imports. Installing them makes the
# registry load and every HTTP/API tool work, and makes `scripts/health_check.py`
# pass. The external OSINT *CLI* tools (sherlock, maigret, holehe, theHarvester,
# exiftool, ...) are installed separately — see bootstrap.sh / SETUP.md — and the
# system degrades gracefully when any of them is absent.
#
# Validated on Python 3.13 (Windows). Floors are chosen so a fresh macOS install
# (Homebrew python3) resolves current wheels. If a specific version ever breaks,
# pin to the exact validated version noted in the comment.
# Install:  python3 -m pip install -r requirements.txt
# ============================================================================
