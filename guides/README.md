# Manual-tooling guides

Step-by-step guides for OSINT capabilities the system can't do automatically (paid, manual,
key-gated, or login-only). The **supervisor writes these during investigations** when it hits
such a gap, so the operator can do the step by hand and feed the result back into the case.

Each guide: what the capability is · best manual tool(s) + URLs · exact steps · how to feed the
result back into the investigation. They double as a training library for the operator.
