# Manual guide: Reverse-image & face search

**Why manual:** every good reverse-image/face engine is paid-API or manual-only (TinEye API is
paid; Yandex/Google Lens/PimEyes have no free automatable API). The system can't do this for you,
but you can do it by hand in a couple of minutes.

**When to use:** you have a profile photo / any image and want to find where else it appears, the
real person behind an avatar, or the original source of an image.

## Best engines (manual, free unless noted)
| Engine | URL | Best at |
|---|---|---|
| **Yandex Images** | https://yandex.com/images/ | **#1 for faces/people** — far better than Google |
| **Google Lens** | https://lens.google.com/ | objects, products, landmarks, text in image |
| **TinEye** | https://tineye.com/ | exact-copy/where-used, oldest-first (find original source) |
| **PimEyes** | https://pimeyes.com/ | face search across the web (free preview; paid to see URLs) |
| **Bing Visual** | https://www.bing.com/visualsearch | second opinion |

## Steps
1. Save the target image (right-click → save, or screenshot the avatar).
2. Go to **Yandex Images** → click the camera icon in the search bar → upload the image.
3. Read results: look for the **same face on other platforms/sites**, a **name**, a **location**,
   or the **original/highest-resolution source**. Yandex clusters similar faces — scan those.
4. Cross-check on **TinEye** (to find the earliest/original posting) and **Google Lens** (for any
   text, logos, or place in the image).
5. For a face with no obvious hits, try **PimEyes** (the free preview shows whether matches exist;
   seeing the source URLs needs a paid plan).
