# Intake drop-zone

Drop incoming OSINT resources here (a tool name, a repo URL, an API doc, a list, a paste) for
the System Manager to triage. The Manager classifies each (CLI / HTTP / website / dead /
duplicate; free / keyed / paid / manual) and decides: **wire-now / backlog / manual-guide /
reject**, recording the decision in `../BACKLOG.md` and `../CHANGELOG.md`.

You can also just paste a resource to the Manager in chat — this folder is for when you want to
queue several for later processing. One file per resource (or a single `queue.md` list).
